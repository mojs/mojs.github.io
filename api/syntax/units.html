# Available Units

The list of available units: px|%|rem|em|ex|cm|ch|mm|in|pt|pc|vh|vw|vmin.

If start and end units of delta are different - mojs will fallback to end units.

Usage with properties:

  // ...
  property :   '20rem',
  property2 :  { '10%' : '100%' },
  // ...
